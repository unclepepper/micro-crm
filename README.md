## Api MICRO CRM


### Что бы запустить приложение, необходимо выполнить команды:


``` make init ``` Для сборки, настройки и запуска контейнеров для приложения

```make m-up ``` Для запуска миграций

``` make test``` Для запуска автотестов приложения
